## Runtime Metaprogramming with React Suspense

**Render Attempt 1**

- Try to read `data.dog` from cache
- Suspend: GraphQL query for `{ dog }`

**Render Attempt 2**

- Try to read `data.dog.displayImage` from cache
- Suspend: GraphQL query for `{ dog { displayImage } }`

**Render Attempt 3**

- Try to read `data.dog.displayImage` from cache
- Success!
